# Hướng Dẫn Sử Dụng AI

Trang web hướng dẫn sử dụng trí tuệ nhân tạo (AI), được xây dựng bằng React và Vite.

## Cài Đặt và Chạy

1. Cài đặt dependencies:
   ```bash
   npm install
   ```

2. Chạy server phát triển:
   ```bash
   npm run dev
   ```

3. Mở trình duyệt và truy cập [http://localhost:5173](http://localhost:5173)

## Xây Dựng

Để xây dựng cho production:
```bash
npm run build
```

## Xem Trước

Sau khi xây dựng, xem trước:
```bash
npm run preview
```

## ESLint

Chạy lint:
```bash
npm run lint
```
